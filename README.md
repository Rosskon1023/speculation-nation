# Speculation NASA

A simple web app to:
- compare and display two cryptocurrencies and their key metrics
- Speculate the potential price of the lower-ranked cryptocurrency if it were to reach the market cap of the higher ranked comparison coin 

## Technologies Used 

- HTML5
- CSS3
- JavaScript/jQuery
- cryptocompare API
